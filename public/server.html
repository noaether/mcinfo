MCINFO

Command List

~help

Shows this help page

~online <server>

Sends the number of players online on the given server

~ping <server>

Shows the ping of the given server

~ip <server>

Sends the direct IP of given server

~all <server>

Sends all the information about given server

~head <player>

Shows the head of given player

~skin <player>

Sends the skin of the given player
